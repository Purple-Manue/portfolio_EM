Explorateur de fichiers
Cet explorateur de fichier affiche en direct le chemin où nous sommes. Il est customisé à la simpson mais n'a aucun rapport avec la série.
La navigation peut se faire soit par le click soit par l'input.
Un bouton retour est aussi prévu pour la navigation.
Upload de fichiers
Cette application permet de charger des fichiers sur une base de données. Vous pouvez vous inscrire ou non.
La seule nuance est le poids des uploads que vous pourrez faire. En cliquant sur la liste des activités, vous pourrez voir alors tous les fichiers que vous avec déjà stockés.
